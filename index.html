MicroGUI

Learning how to program touch displays as well as creating good looking and functional graphical user interfaces (GUI) is not the easiest of tasks. MicroGUI was created to lower the entry barrier for those who want to implement displays in their embedded projects.

MicroGUI is an open-source toolset for easily creating GUIs and quickly deploying them on embedded displays.

Safe Remote Control Solution for MARV with 5G

MARV, short for Marine Autonomous Reasearch Vehicle, is a modified personal watercraft that is used in many research projects at Chalmers. It was of interest to know whether a robust and fast data link could be established between MARV and a computer stationed on land.

A remote control solution with real-time camera feed was implemented for MARV to demonstrate the possibilities of 5G.

Automatic Robot Garage

Automatic Robot Garage is a fun project that I started with my father to create a fancy garage to house his Husqvarna Automower. He built the miniature garage and I put together the electronics and software for it.

The garage features a motor-controlled port, facade lighting and a connection to Husqvarna's cloud for live updates of the Automower's state.
