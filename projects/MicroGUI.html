MicroGUI

Summer 2022



Description


Learning how to program touch displays as well as creating good looking and functional graphical user interfaces (GUI) is not the easiest of tasks. MicroGUI was created to lower the entry barrier for those who want to implement displays in their embedded projects.

MicroGUI is an open-source toolset for easily creating GUIs (in a web application) and quickly deploying them on embedded displays (with an Arduino library).

The work that I did for MicroGUI was mainly developing the Arduino library (MicroGUI-Embedded) for rendering the GUIs on embedded displays, but I also implemented a few things in the web application.


If you are interested in learning more about what MicroGUI has to offer, just click the icon to the right and browse the repositories for the web application and the embedded library.
