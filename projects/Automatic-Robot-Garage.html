Automatic Robot Garage

Spring 2021 - Fall 2022

Overview


Automatic Robot Garage is a fun project that I started with my father to create a fancy garage to house his Husqvarna Automower. He built the miniature garage and I put together the electronics and software for it.

The garage features a motor-controlled port, facade lighting and a connection to Husqvarna's cloud for live updates of the Automower's state.
